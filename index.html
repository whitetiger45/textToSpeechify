Text to Speechify

Usage: textToSpeechify [-h] [--version] -f HTML_FILE [-hfp] [-O OUT_FILE]

Prepare text for speech.


optional arguments:

-h, --help show this help message and exit

--version show program's version number and exit

-f HTML_FILE, --file HTML_FILE

html file to be parsed

-hfp, --htmlfrompdf html file generated from a pdf file using pdftohtml

-O OUT_FILE, --output OUT_FILE

file to write output to
